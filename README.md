# SPREAD-NEWS

Welcome to **SPREAD-NEWS**, a modern news aggregation platform designed to deliver the latest news directly to your inbox, seamlessly integrated with Mailchimp. This project leverages contemporary web technologies to create a responsive, user-friendly interface, ensuring an optimal experience across all devices.

## Features
- **News Aggregation**: Collects the latest articles from various sources, making it easier to stay informed.
- **Mailchimp Integration**: Effortlessly connect with Mailchimp to send curated news updates to subscribers.
- **Responsive Design**: Built with Bootstrap to ensure accessibility and usability across devices of all sizes.
- **Elegant Typography**: Utilizes Schoolbell and Comic Neue fonts for a pleasant reading experience.

## Getting Started
1. **Clone the repository**:
   ```
   git clone https://github.com/GhostAssa/SPREAD-NEWS.git
   ```
2. **Install dependencies**:
   Navigate to the project directory and run:
   ```
   npm install  
   ```
3. **Set up the Mailchimp API**:
   Follow [Mailchimp API documentation](https://mailchimp.com/developer/) to set up your account and obtain your API key.
4. **Run the application**:
   ```
   npm start
   ```
5. Visit `http://localhost:3000` in your browser to see the application in action.
